# User Onboarding Guide

1. **Account Creation**
   - Register via landing page or invite email.
   - Verify email + enable MFA.
2. **Connect Brokerage**
   - Supported: Zerodha, AngelOne.
   - Use OAuth pop-up and grant read permissions.
3. **Configure Workspace**
   - Create watchlists, upload CSV holdings.
   - Assign roles (viewer/editor/admin).
4. **Activate AI Assistant**
   - Accept usage disclaimer.
   - Toggle “Grounding required” in settings.
5. **Upgrade Plan**
   - Choose Basic/Pro/Enterprise.
   - Pay via Razorpay; invoice emailed automatically.

Support: support@finbot.in (8am–8pm IST).
